# 라면 사기
- N개의 라면 공장
- i 라면 공장에서 Ai 개의 라면 구매
```
1. i번 공장에서 라면 하나 구매. 비용 3원
2. i번 공장과 i+1 공장에서 라면 하나씩 구매 비용 5원
3. i, i+1. i+2 공장에서 라면 하나씩 비용 7
```
### 문제 해결
3 -> 2-> 1 순으로 최선
i, i+1, i+2 min 구하고 min * 7 추가 i, i+1, i+2 만큼 빼기
i, i+1, min 구하고 min * 5 추가 i, i+1 에 i 만큼 빼기
i * 3 추가 

마지막 두 개
min 구하고 * 5
남은 거 합해서 * 3

### 예외 사항
- 2 3 2 1 
```
1 1
1 1 1 
  1 1 1
```
